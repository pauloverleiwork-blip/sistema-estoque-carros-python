# Sistema de Estoque de Veículos

Projeto desenvolvido em Python para gerenciamento de veículos via terminal, com persistência de dados em JSON.

## Funcionalidades
- Adicionar veículos  
- Listar veículos  
- Deletar veículos  
- Filtrar por marca e preço  
- Alterar preço  

## Tecnologias
- Python  
- JSON  

## Objetivo
Este projeto foi desenvolvido com o objetivo de demonstrar minha evolução prática na programação, aplicando conhecimentos adquiridos ao longo dos estudos. A proposta é aprimorá-lo continuamente, servindo como evidência do meu desenvolvimento técnico e aprendizado constante.
